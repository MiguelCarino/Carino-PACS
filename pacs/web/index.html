Print receiver

Point a print-only modality at this AE / port as its DICOM printer. Each captured film is queued in 📎 Pending for you to identify, then Approve to convert to DICOM and forward.

Emergency failover

Tick a destination's 🚨 box (in Destinations) to mark it a primary. When it's unreachable past the threshold you get a pop-up to activate the emergency worklist; studies received meanwhile are held and back-filled when it returns.

Modality Worklist (MWL)

Point a modality at this AE / port as its Modality Worklist source. It pulls every open order (from 📋 Orders) by C-FIND; set an order's Target modality AE to steer it to one station. The order's Study UID is burned into the exam, so the returned study reconciles exactly.

Emergency RIS

Point a RIS (or an HL7 test sender) at this host:port over MLLP to feed ORM^O01 orders, or hand-key them in 📋 Orders. Tell the tech to type the Accession # into the modality — that's what matches the study back to its order. Image delivery is never blocked by a missing match.

Integrations

Set this to where you serve DICOM-editor. When filled, each study in History gets an ✎ Edit tags button that opens the study straight in the editor.
